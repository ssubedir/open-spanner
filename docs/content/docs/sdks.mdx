Open Spanner publishes generated SDKs from the OpenAPI spec.

| Language | Package | Install |
| --- | --- | --- |
| Go | `github.com/ssubedir/open-spanner/sdk/go` | `go get github.com/ssubedir/open-spanner/sdk/go@latest` |
| TypeScript | `@ssubedir/open-spanner` | `npm install @ssubedir/open-spanner` |
| Python | `open-spanner` | `pip install open-spanner` |
| C# | `OpenSpanner` | `dotnet add package OpenSpanner` |

## Generated From OpenAPI

SDKs are generated from the SDK-friendly OpenAPI spec in `openapi/sdk-openapi.json`.

When the API shape changes, regenerate SDKs before publishing package releases.

```sh
task sdk:typescript:check
task sdk:python:check
task sdk:csharp:check
task sdk:go:check
```
