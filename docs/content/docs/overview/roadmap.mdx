---
title: Roadmap
description: Direction for upcoming Open Spanner work.
---

This roadmap is directional. It is meant to show product intent, not promise exact release dates.

## Near Term

- Registration controls for deployments that do not want open sign-up.
- API key naming, rotation, and deletion polish.
- More dashboard views around ingestion activity and alert history.
- More focused SDK examples for common backend patterns.

## Product Hardening

- API key scopes for meter and usage operations.
- Role-aware dashboard access.
- Better operational observability for pruning, ingestion, and storage readiness.
- Additional alert destination types beyond webhooks.
- Import and backfill workflows for historical usage.

## Possible Later Work

- Streaming ingestion for very high volume workloads.
- WebSocket support for live dashboard updates and usage activity streams.
- gRPC support for service-to-service clients that need a typed protocol beyond REST.
- Queue-based ingestion for asynchronous usage pipelines.
- Export cleanup policies for generated files.
- Additional export destinations beyond CSV.
- More storage and deployment packaging options.
