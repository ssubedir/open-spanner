title: Python
description: Use Open Spanner from Python services and jobs.
---

Install:

```sh
pip install open-spanner
```

## Runnable Example

Use the repository example as the canonical Python sample:

- Example project: [`examples/rest/basic/python`](https://github.com/ssubedir/open-spanner/tree/main/examples/rest/basic/python)
- Source file: [`examples/rest/basic/python/basic.py`](https://github.com/ssubedir/open-spanner/blob/main/examples/rest/basic/python/basic.py)

```sh
cd examples/rest/basic/python
uv run python basic.py
```

Create an API key in the dashboard first, then replace the `osp_...` placeholder in `basic.py`. Set `OPEN_SPANNER_BASE_URL` when your service is not running at `http://localhost:18081`.
