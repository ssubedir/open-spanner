---
title: Go
description: Use Open Spanner from Go services.
---

Install:

```sh
go get github.com/ssubedir/open-spanner/sdk/go@latest
```

## Runnable Example

Use the repository example as the canonical Go sample:

- Example project: [`examples/go/basic`](https://github.com/ssubedir/open-spanner/tree/main/examples/go/basic)
- Source file: [`examples/go/basic/main.go`](https://github.com/ssubedir/open-spanner/blob/main/examples/go/basic/main.go)

```sh
cd examples/go/basic
go run main.go
```

Create an API key in the dashboard first, then replace the `osp_...` placeholder in `main.go`. Set `OPEN_SPANNER_BASE_URL` when your service is not running at `http://localhost:18081`.
