---
title: TypeScript
description: Use Open Spanner from Node.js or TypeScript services.
---

Install:

```sh
npm install @ssubedir/open-spanner
```

## Runnable Example

Use the repository example as the canonical TypeScript sample:

- Example project: [`examples/basic/typescript`](https://github.com/ssubedir/open-spanner/tree/main/examples/basic/typescript)
- Source file: [`examples/basic/typescript/basic.ts`](https://github.com/ssubedir/open-spanner/blob/main/examples/basic/typescript/basic.ts)

```sh
cd examples/basic/typescript
npm install
npm run start
```

Create an API key in the dashboard first, then replace the `osp_...` placeholder in `basic.ts`. Set `OPEN_SPANNER_BASE_URL` when your service is not running at `http://localhost:18081`.
