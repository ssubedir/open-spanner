---
title: C#
description: Use Open Spanner from .NET services.
---

Install:

```sh
dotnet add package OpenSpanner
```

## Runnable Example

Use the repository example as the canonical C# sample:

- Example project: [`examples/basic/csharp`](https://github.com/ssubedir/open-spanner/tree/main/examples/basic/csharp)
- Source file: [`examples/basic/csharp/Program.cs`](https://github.com/ssubedir/open-spanner/blob/main/examples/basic/csharp/Program.cs)

```sh
cd examples/basic/csharp
dotnet run --project OpenSpanner.Example.csproj
```

Create an API key in the dashboard first, then replace the `osp_...` placeholder in `Program.cs`. Set `OPEN_SPANNER_BASE_URL` when your service is not running at `http://localhost:18081`.
