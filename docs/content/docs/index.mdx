title: Open Spanner

Open Spanner is a small metering service for recording usage events and querying usage buckets.
It is designed around four questions:

- Who used something?
- When did it happen?
- How much was used?
- Where should that usage be grouped or filtered?

The API is intentionally simple at the edge, while the internals keep domain, application, and adapter code separate so storage and SDK generation can evolve without rewriting the product.

## Main Surfaces

- API: `/v1`
- Swagger UI: `/docs`
- Dashboard: `/overview`, `/meters`, `/usage`
- OpenAPI spec: `openapi/openapi.json` in the repository

## Local First

Start with SQLite for development:

```sh
task run:sqlite
```

Then open the dashboard or API docs in your browser.
