# Go Basic Example

Run this example:

```sh
go run main.go
```

This example uses the Go SDK from this checkout. Create an API key in the
dashboard first, then replace `osp_...` in `main.go`.
Set `OPEN_SPANNER_BASE_URL` when your Open Spanner API is not running at
`http://localhost:18081`.

Advanced use-case examples live under `../../advance`.
