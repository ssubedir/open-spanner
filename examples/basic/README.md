# Basic SDK Examples

These examples show the smallest useful SDK flow: create one meter, write one
usage event, and see dimension validation reject invalid metadata.

| SDK | Command |
| --- | --- |
| TypeScript | `cd examples/basic/typescript && npm install && npm run start` |
| Python | `cd examples/basic/python && uv run python basic.py` |
| C# | `cd examples/basic/csharp && dotnet run --project OpenSpanner.Example.csproj` |
| Go | `cd examples/basic/go && go run main.go` |

Create an API key in the dashboard before running an example. Advanced
use-case examples live under `../advance`.
