# API Request Metering

Track request volume by customer, endpoint, method, status, and region. This is
the starting point for billing on API calls, monitoring customer traffic, and
finding high-volume routes.

| SDK | Command |
| --- | --- |
| TypeScript | `npm install && npm run start` |
| Python | `uv run python python.py` |
| C# | `dotnet run --project OpenSpanner.ApiRequests.csproj` |
| Go | `go run .` |
