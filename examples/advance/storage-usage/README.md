# Storage Usage

Track capacity over time by storage tier, region, and resource type. This style
works for GB-hours, object storage, backups, and archive tiers.

| SDK | Command |
| --- | --- |
| TypeScript | `npm install && npm run start` |
| Python | `uv run python python.py` |
| C# | `dotnet run --project OpenSpanner.StorageUsage.csproj` |
| Go | `go run .` |
