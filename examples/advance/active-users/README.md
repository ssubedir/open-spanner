# Active Users

Track active seats or users by customer plan, workspace type, and region. This
fits SaaS products where usage is measured by participation over a billing
period.

| SDK | Command |
| --- | --- |
| TypeScript | `npm install && npm run start` |
| Python | `uv run python python.py` |
| C# | `dotnet run --project OpenSpanner.ActiveUsers.csproj` |
| Go | `go run .` |
