# AI Token Usage

Track token consumption by model, provider, operation, and cache behavior. This
fits AI products that need usage visibility before pricing, rate limits, or
cost controls are layered on top.

| SDK | Command |
| --- | --- |
| TypeScript | `npm install && npm run start` |
| Python | `uv run python python.py` |
| C# | `dotnet run --project OpenSpanner.AiTokens.csproj` |
| Go | `go run .` |
