# Background Jobs

Track asynchronous work by queue, result status, and worker region. This is for
imports, exports, scheduled work, retries, and other processing pipelines.

| SDK | Command |
| --- | --- |
| TypeScript | `npm install && npm run start` |
| Python | `uv run python python.py` |
| C# | `dotnet run --project OpenSpanner.BackgroundJobs.csproj` |
| Go | `go run .` |
