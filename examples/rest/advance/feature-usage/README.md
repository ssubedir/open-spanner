# Feature Usage

Track premium feature, add-on, or entitlement usage by plan and source. This is
useful for product analytics, packaging, and billing experiments.

| SDK | Command |
| --- | --- |
| TypeScript | `npm install && npm run start` |
| Python | `uv run python python.py` |
| C# | `dotnet run --project OpenSpanner.FeatureUsage.csproj` |
| Go | `go run .` |
